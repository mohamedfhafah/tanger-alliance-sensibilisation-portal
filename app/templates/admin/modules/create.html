Guide de création

Titre du module

Choisissez un titre clair et descriptif pour votre module de formation.

Description

Résumé bref du contenu et des objectifs d'apprentissage du module.

Contenu

Le contenu principal du module. Vous pouvez utiliser:

  • HTML pour la mise en forme
  • Images et vidéos
  • Listes et tableaux
  • Liens externes
Ordre

Détermine la position du module dans la liste. Les modules sont affichés par ordre croissant.

Après la création

Une fois le module créé, vous pourrez:

  • Ajouter un quiz d'évaluation
  • Modifier le contenu
  • Réorganiser l'ordre des modules
  • Activer/désactiver le module
