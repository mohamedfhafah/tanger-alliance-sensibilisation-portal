Adaptable aux mobiles, tablettes et ordinateurs

Test d'Accessibilité

Utilisez la touche Tab pour naviguer entre les éléments. Les éléments focusés doivent avoir un indicateur visuel clair.

Instructions de Test

  1. Test Responsive : Redimensionnez la fenêtre pour tester différentes tailles d'écran
  2. Test Mobile : Cliquez sur le bouton hamburger en mode mobile
  3. Test Sidebar : Testez l'expansion/contraction de la sidebar
  4. Test Accessibilité : Utilisez Tab et les flèches pour naviguer au clavier
  5. Test Recherche : Testez la fonction de recherche dans la barre de navigation
  6. Test Menus : Testez l'expansion des sous-menus dans la sidebar
